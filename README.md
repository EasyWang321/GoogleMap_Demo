# GoogleMap Demo

展示一些常用的Google地图操作。

- 获取我的位置。/ get my location.
- 画自定义图形。/ draw custom polygons.
- 计算自定义图形的中心
- 计算自定义图形的重心
- 判断一个点是否在自定义图形内部
- 多级锚点悬浮滑动面板
